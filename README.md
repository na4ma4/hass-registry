# Home Assistant Add-ons: na4ma4

[![License](https://img.shields.io/github/license/na4ma4/hass-registry)](LICENSE)

This repository contains Home Assistant add-ons maintained by na4ma4.

## Add-ons

This repository includes the following add-ons:

### [Meshtastic: Protobuf to JSON](./meshtastic-mqtt-translate)

![Version](https://img.shields.io/badge/version-0.2.2-blue)
![Supports aarch64 Architecture](https://img.shields.io/badge/aarch64-yes-green)
![Supports amd64 Architecture](https://img.shields.io/badge/amd64-yes-green)
![Supports armv7 Architecture](https://img.shields.io/badge/armv7-yes-green)

Convert Meshtastic protobuf messages to JSON and publish them back to MQTT.

## Installation

1. Navigate to the **Settings** → **Add-ons** → **Add-on Store** in your Home Assistant instance
2. Click the three-dot menu in the top right corner and select **Repositories**
3. Add the following repository URL:
   ```
   https://github.com/na4ma4/hass-registry
   ```
4. The add-ons from this repository should now appear in your add-on store
